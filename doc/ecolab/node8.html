Compiling and running Eco Lab for Aqua

Build Eco Lab with AQUA=1. In the models directory, an ``application bundle'' is created for each model, for example ecolab.app, using the utility script mkmacapp. If you try to run Eco Lab scripts in the usual way, you will get an error message: 

SetFrontProcess failed,-606
Instead, you must invoke Eco Lab in a rather cumbersome way: 
ecolab.app/Contents/MacOS/ecolab ecolab.tcl
To make it a little easier, the macrun script is provided to do this for you -- try 
macrun jellyfish.tcl lakes/OTM
